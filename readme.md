# ARTDM-171 Assignments

This repository contains coursework and assignments for the ARTDM-171 Web Design class. Each assignment and practice project is organized into its own folder, with subfolders for mini websites and related resources.

> [!NOTE]
> Will be updating the filesystem & contents to reflect proper structure -- i.e. Challenges, Practices, etc. folders for a cleaner maintenance and navigation experience.
> This should have been included in the assignments from the beginning...

## Project Structure

- **challengeX/**: Challenge assignments (e.g., challenge1, challenge2, etc.)
- **practiceX/**: Practice assignments (e.g., practice1, practice2, etc.)
- **mini_website/**: Subfolders for multi-page mini website projects within challenges or practices
- **index.html**: Homepage with a sidebar navigation to all assignments and projects

## Navigation

- The main `index.html` file serves as a homepage and navigation hub.
- Use the collapsible sidebar to browse all challenges and practices.
- Each folder contains its own `index.html` and related files.
- For mini websites, expand the relevant section in the sidebar to access all pages.

## Viewing on GitHub

- Browse the repository tree to view folders and files.
- Click on any HTML file to preview its code.
- To view the site live, use GitHub Pages if enabled, or open the HTML files locally in your browser.
